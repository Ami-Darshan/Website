### Description
Personal Portfolio
### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap
- Edit Content From One Place

### Setup

Get the code
 
Install required dependencies 

<pre>yarn install</pre>
or
<pre>npm install</pre>


Start the server

<pre>yarn start</pre>
<pre>npm run start</pre>

### More

Modify pages content in  `src/content_option.js`.
